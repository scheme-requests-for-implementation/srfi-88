** by Marc Feeley

This repository hosts [[http://srfi.schemers.org/srfi-88/][SRFI 88]]: Keyword objects, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-88/][join the SRFI-88 mailing list]] and send your message there.
